# CivicQ Candidate Portal - Features Overview

## What Politicians Can Now Do

### 1. View Questions Voters Are Asking

**Dashboard View**: Shows all questions ranked by voter interest
- See question text and context
- View issue tags (housing, economy, transportation, etc.)
- See upvotes and rank scores
- Filter by answered/unanswered status

**Question Details**: Full context for each question
- Additional background information
- Community engagement metrics
- Related issues and topics

**Professional Recording Interface**:
- ✅ Browser-based recording (no app install needed)

- ✅ Retake as many times as needed
- ✅ Upload progress tracking

**Recording Features**:
- Automatic duration tracking

## Key User Flows

### Answering a Question

1. **Login** to candidate account
2. **Navigate** to Dashboard
3. **View** pending questions list
4. **Click** "Answer" on any question
5. **Read** question details and context
6. **Start** camera and grant permissions
7. **Record** your answer (up to 2 minutes)
8. **Preview** the recording
9. **Submit** or retake if needed
10. **Return** to dashboard with updated stats

### Editing Profile

1. **Click** "Edit Profile" in dashboard or navigation
2. **Update** any information fields
3. **Add** campaign priorities or endorsements
4. **Save** changes
5. **View** updated public profile

### First-Time Setup

1. **Receive** invitation email (or visit onboarding page)
2. **Complete** 4-step onboarding flow
3. **Verify** candidacy with filing ID
4. **Set up** profile information
5. **Start** answering questions immediately
